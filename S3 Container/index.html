CodeMeet is a platform created to be used by a Developers group. If you are a member, or even the Manager that coordinates the development, you will find helpful a tool like CodeMeet :

+ Whoever can post a ticket, which is a signalation of a problem occuring in a specific project, as it could be an error about networking, an alert about missing permissions or security warning or even a simple bug that happens under certain circumstances. When doing so, the team would appreciate you using more details about the problem - like indicating the exact file and line in which the error occurs.
You can also an example, like a stack overflow thread that explains the same problem or explains a possible solution.

+ As a developer you can solve this tickets that yourself or your coworkers created, posting a comment (also a link, is there is one, of what helped you through the problem solving!) if you feel one should be needed.
